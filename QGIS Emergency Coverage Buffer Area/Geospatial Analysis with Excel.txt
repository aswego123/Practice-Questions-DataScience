# ‚úÖ **Step-by-Step (Excel)**

### **1. Load the data**

* Open Excel ‚Üí **Data ‚Üí From Text/CSV**
* Import the table into a sheet.

### **2. Add Population_per_OurStore**

Insert a new column:

```
Population_per_OurStore = Population / Our_Stores
```

Calculate for each row.

### **3. (Optional) Add Coverage_Ratio**

```
Coverage_Ratio = Our_Stores / (Our_Stores + Competitor_Stores)
```

Useful for map visuals.

### **4. Sort descending**

Select the full table ‚Üí **Sort ‚Üí Sort by Population_per_OurStore ‚Üí Largest to Smallest**

This reveals where MetroBeans is most under-served.

### **5. Map the hotspot (optional)**

Use:

* **Insert ‚Üí Maps ‚Üí Filled Map**
  or
* **Insert ‚Üí 3D Maps**

Plot:

* **Population_per_OurStore**
* **Coverage_Ratio**

The top neighborhood should appear as the strongest unmet demand.
